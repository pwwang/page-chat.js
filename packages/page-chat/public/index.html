Page Chat

An in-page AI chatbot that reads your web page and answers questions about it.

Client-side only No backend required File attachments Screenshot support

About this demo

Page Chat is loaded as an IIFE script on this page. It automatically reads the page content and lets you ask questions about it.

You can also attach local files (PDF, images, text) or take a screenshot to include as context in your questions.

Things to try

  • Ask "What is this page about?"
  • Ask "Summarize the content on this page"
  • Click the paperclip icon to attach a local file and ask questions about it
  • Click the camera icon to take a screenshot and include it as context
  • Click the rescan icon to re-read the page content if you navigate or scroll

Quick integration

Add Page Chat to any page with a single script tag:

<script src="https://cdn.jsdelivr.net/npm/page-chat/dist/iife/page-chat.demo.js"
        crossorigin="true"></script>

Or configure with your own LLM:

import { PageChat } from 'page-chat'

const chat = new PageChat({
  model: 'gpt-4o-mini',
  baseURL: 'https://api.openai.com/v1',
  apiKey: 'YOUR_API_KEY',
  persist: true // optional, enable conversation history persistence across page reloads
})
chat.panel.show()

Sample content for the chatbot

Page Chat uses a text-based DOM extraction approach — no screenshots needed for page reading. It extracts headings, paragraphs, links, and interactive elements into a structured representation that the LLM can reason about efficiently.

The chatbot maintains a multi-turn conversation history, so you can ask follow-up questions naturally. Use the clear button to start a fresh conversation.

Supported file types for attachment: plain text, Markdown, CSV, JSON, JavaScript, TypeScript, PDF, and images (PNG, JPG, GIF, WebP).
